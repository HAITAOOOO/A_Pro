一些说明：

1.为避免6020与3508的0x1FF冲突，只应用了以0x2FF为控制报文的后三个ID
2.采用两个结构体一个程序的方式实现串级
3.将pid初始化与pid计算从一个函数拆开，好处在于减少了pid重复初始化的时间，
  坏处是每一个pid结构数据不能被覆盖，即有一个电机必须初始化一个结构体，增大了内存占用
4.引入系统
