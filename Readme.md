1.为避免 6020 与 3508 的 0x1FF 冲突，只应用了以 0x2FF 为控制报文的后三个 ID 2.采用两个结构体一个程序的方式实现串级 3.将 pid 初始化与 pid 计算从一个函数拆开，好处在于减少了 pid 重复初始化的时间，
坏处是每一个 pid 结构数据不能被覆盖，即有一个电机必须初始化一个结构体，增大了内存占用 4.引入系统
